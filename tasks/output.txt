userController.ts:
- addUserRoles: Ajoute des rôles à un utilisateur
- removeUserRoles: Supprime un rôle spécifique d'un utilisateur
- forgotPassword: Gère la demande de réinitialisation de mot de passe
- resetPassword: Effectue la réinitialisation du mot de passe
- getRegistrationsByUserId: Récupère toutes les inscriptions d'un utilisateur
- getPaymentsByUserId: Récupère tous les paiements d'un utilisateur

paymentController.ts:
- createSubscription: Crée un abonnement mensuel via Stripe
- createDonation: Traite un don unique via Stripe
- updatePaymentMethod: Met à jour la méthode de paiement dans Stripe
- cancelSubscription: Annule un abonnement existant
- handleWebhook: Gère les webhooks Stripe
- handleSuccessfulPayment: Traite les paiements réussis (méthode privée)
- handleFailedPayment: Traite les paiements échoués (méthode privée)

itemController.ts:
- getItemByStatus: Récupère les items selon leur statut
- addCategoryToItem: Ajoute une catégorie à un item
- getAllCategoriesOfItem: Récupère toutes les catégories d'un item
- deleteCategoryOfItem: Supprime une catégorie d'un item

eventController.ts:
- getAllEventRegistrations: Récupère toutes les inscriptions pour un événement

categoryController.ts:
- createChildCategory: Crée une sous-catégorie
- getAllCategoriesOfCategory: Récupère toutes les sous-catégories d'une catégorie de manière récursive
- getCategories: Fonction utilitaire récursive pour récupérer la hiérarchie des catégories

authController.ts:
- login: Gère l'authentification et envoie un OTP
- register: Inscrit un nouvel utilisateur
- verifyOTP: Vérifie le code OTP et génère un JWT
- getCurrentUser: Récupère l'utilisateur actuellement connecté
- revokeAllTokens: Révoque tous les tokens JWT
- revokeUserTokens: Révoque les tokens d'un utilisateur spécifique
- isTokenRevoked: Vérifie si un token est révoqué
